#简书项目开发01

标签（空格分隔）： js

简书项目开发01
新建项目
1.已安装react的情况下，在桌面 shift + 右键 打开服务器；
2.输入命令 create-react-app +项目名称 jianshu，新建项目。
3.如果安装失败，排除网络错误之后，就换成cnpm create-react-app 命令来安装
4.启动项目： 进入文件夹：输入cd jianshu 进入文件夹后输入yarn start
项目代码：
1.为了避免直接引入css样式文件造成样式混乱，安装使用components,它可以使组件的样式只对自己生效。
components安装命令：yarn add styled-components
2.为了适应所有电脑的所有浏览器，让代码在所有浏览器上表现一致，需要一个reset.css文件，用以统一所有的默认配置。
网址https://meyerweb.com/eric/tools/css/reset/ 将的代码复制下来放入全局样式中
3.为了支持动画效果，需要安装动画模块，安装命令 yarn add react-transition-group；
4.使用时引入，注意CSS大写，格式为 import { CSSTransition } from ‘react-transition-group’;
5.安装redux，安装命令：yarn add redux 。
6.在react中使用redux，需要安装，安装命令为：yarn add react-redux
7.immutable对象，Facebook历时三年开发的库，是生成不可修改的对象用的
安装命令：yarn add immutable
8.安装redux-immutable
安装命令：yarn add redux-immutable
9.安装thunk中间件
安装命令：yarn add redux-thunk
10.安装第三方模块，用以请求数据。
安装命令 ：yarn add axios

遇到问题：
1.css文件引入一次，会全局使用，会造成样式混乱，不建议直接引入css文件
